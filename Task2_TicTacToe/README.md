# 🎮 Tic-Tac-Toe AI — CODSOFT Task 2

An **unbeatable AI** that plays Tic-Tac-Toe against a human player using **Minimax with Alpha-Beta Pruning**.

## How to Run

Just open `index.html` in your browser. That's it! No installation needed.

```
start index.html
```

## Algorithm

| Component | Description |
|-----------|-------------|
| **Minimax** | Recursively explores all possible moves to find the optimal play |
| **Alpha-Beta Pruning** | Skips irrelevant branches for faster computation |
| **Depth Penalty** | AI prefers winning sooner rather than later |

## Difficulty Levels

| Level | Behavior |
|-------|----------|
| 😊 Easy | Random moves |
| 🤔 Medium | 50% smart, 50% random |
| 🤖 Unbeatable | Full Minimax — AI never loses |

## Tech Stack

- **HTML5** — Game structure
- **CSS3** — Dark theme with animations
- **Vanilla JavaScript** — Minimax AI + game logic
